Preguntas Frecuentes - Uso¶

Respuestas a las preguntas más comunes sobre el uso de Jackent.

General¶

¿Puedo usar Jackent en móvil?¶

Sí, la interfaz es responsive. Funciona en:

  • ✅ Navegadores móviles (Chrome, Safari)
  • ✅ Tablets
  • ❌ No hay app nativa (aún)

¿Es necesario tener VPN?¶

Recomendado para descargas P2P:

  • ✅ Oculta tu IP ante trackers
  • ✅ Evita problemas con ISP
  • ✅ Mayor privacidad

No es obligatorio, pero sí recomendable para uso intensivo.

¿Cuánto espacio necesito?¶

Depende de tu uso:

  • Mínimo: 10-20 GB
  • Recomendado: 50-100 GB
  • Uso intensivo: 500 GB+

Configura auto-eliminación y límite de espacio en Jackent.

Búsqueda¶

¿Por qué no aparecen resultados?¶

Posibles causas:

  1. Trackers no seleccionados - Marca al menos un tracker
  2. Jackett no configurado - Verifica que Jackett tenga trackers activos
  3. Búsqueda muy específica - Prueba términos más generales
  4. Trackers caídos - Algunos trackers pueden estar temporalmente inactivos

Solución: 

1. Verificar indicadores de estado (deben estar en verde)
2. Probar con nombre en inglés
3. Simplificar búsqueda (solo título, sin año ni calidad)

¿Cómo busco series por episodio específico?¶

Usa el formato estándar:

Breaking Bad S01E01        # Temporada 1, Episodio 1
Game of Thrones S05E09     # Temporada 5, Episodio 9
The Office US S02          # Temporada 2 completa

¿Qué trackers son los mejores?¶

Depende del contenido:

Tipo Trackers Recomendados
Películas 1337x, YTS, RARBG
Series EZTV, ThePirateBay
Anime Nyaa, AnimeTosho
Música Rutracker, ThePirateBay
Software 1337x (solo open source)
Documentales MVGroup, DocsPedia

qBittorrent y Descargas¶

¿Cuánto tarda en descargarse un torrent?¶

Depende del tamaño y la disponibilidad:

  • Archivos pequeños (< 1 GB): 5-30 minutos
  • Películas HD (1-5 GB): 30-90 minutos
  • Packs grandes (> 10 GB): 1-4 horas

Tiempos estimados para conexiones de 50 Mbps

¿Por qué algunos torrents no descargan?¶

Causas comunes:

  1. Sin seeders - El torrent está muerto
  2. Puertos bloqueados - Firewall o router
  3. Límite de velocidad - Configuración restrictiva

Solución:

  • Busca otro torrent con más seeders (> 50)
  • Verifica puertos en qBittorrent
  • Revisa configuración de firewall

¿Cómo libero espacio en disco?¶

Opción 1: Auto-eliminación en Jackent

  1. Clic en ⚙️ (Configuración)
  2. Activar "Auto-eliminar torrents completados"
  3. Configurar horas (ej: 24 horas)

Opción 2: Manual en qBittorrent

  1. Abrir http://localhost:8080
  2. Seleccionar torrents completados
  3. Clic derecho > Eliminar (con archivos)

Tip: Configura límite de espacio en Jackent para gestión automática.

¿Dónde se guardan los archivos?¶

En la carpeta Descargas del proyecto:

Jackent/
└── Descargas/
    ├── The.Matrix.1999/
    ├── Breaking.Bad.S01/
    └── ...

Windows: C:\Users\TU_USUARIO\Desktop\Jackent\Descargas

Linux/Mac: /home/usuario/Jackent/Descargas

Problemas Comunes¶

El indicador de estado está en rojo¶

Causas:

  1. Backend no iniciado - Docker no está corriendo
  2. Jackett caído - Servicio no disponible
  3. qBittorrent offline - Contenedor detenido

  4. Configuración incorrecta - Variables de entorno mal configuradas

  5. Backend no iniciado - docker compose ps para verificar

  6. API Key incorrecta - Revisar .env
  7. Jackett no accesible - Verificar http://localhost:9117

Solución: 

# Verificar logs
docker compose logs backend

# Reiniciar servicios
docker compose restart backend

No puedo añadir torrents a qBittorrent¶

Verificar:

  1. ✅ Credenciales válidas en .env
  2. ✅ qBittorrent está corriendo
  3. ✅ Tienes espacio en disco disponible
  4. ✅ El torrent tiene seeders

Test manual: 

curl -i --header "Referer: http://localhost:8080" --data "username=admin&password=tu_password" http://localhost:8080/api/v2/auth/login

Si responde con Ok., las credenciales son correctas.

La búsqueda es muy lenta¶

Optimizaciones:

  1. Reduce el número de trackers (máx 3-4)
  2. Limita resultados a 30-50
  3. Activa "Solo con Seeders" Solución:
# 1. Verificar servicios
docker compose ps

# 2. Ver logs
docker compose logs backend
docker compose logs jackett
docker compose logs qbittorrent

# 3. Verificar .env
cat .env

# 4. Reiniciar servicios
docker compose restart

No aparecen mis torrents en qBittorrent¶

Verificar:

  1. ¿Conectividad con qBittorrent?
  2. ¿Credenciales correctas en .env?
  3. ¿Puertos accesibles?
# Verificar acceso a qBittorrent
curl http://localhost:8080

# Ver logs
docker compose logs qbittorrent

Las descargas van muy lentas¶

Optimizar:

  1. Elegir torrents con más seeders
  2. Configurar puertos en router (port forwarding)
  3. Ajustar límites en qBittorrent Web UI
  4. Verificar que Jackett esté actualizado

Privacidad y Seguridad¶

¿Es seguro usar Jackent?¶

Sí, Jackent es seguro:

  • ✅ No almacena contraseñas ni datos personales
  • ✅ Configuración solo se guarda localmente (.env)
  • ✅ Todo funciona en tu servidor local
  • ✅ No hay tracking ni analytics externos
  • ✅ Código abierto y auditable

¿Necesito usar VPN?¶

Sí, es recomendable para descargas P2P:

Usa VPN para:

  • Ocultar tu IP ante trackers
  • Evitar problemas con tu ISP
  • Mayor privacidad en general
  • Acceder a trackers bloqueados

VPNs recomendadas: ProtonVPN, Mullvad, NordVPN

¿Mis archivos son privados?¶

Sí, totalmente:

  • Todo se almacena en tu servidor local
  • Sin servicios en la nube
  • Tú controlas tus datos
  • Nadie más tiene acceso

Avanzado¶

¿Cómo integro Jackent con Plex/Jellyfin?¶

Método recomendado:

1. Configurar biblioteca de Plex/Jellyfin apuntando a ./Descargas
2. Buscar contenido en Jackent
3. Añadir a qBittorrent
4. Esperar a que termine la descarga
5. Plex/Jellyfin detectará automáticamente los nuevos archivos
6. Escanear biblioteca

Ejemplo de rutas: 

Películas: /ruta/a/Jackent/Descargas/Peliculas
Series: /ruta/a/Jackent/Descargas/Series

¿Puedo automatizar descargas?¶

Sí, usando scripts y la API de qBittorrent:

# Script ejemplo para añadir magnet automáticamente
curl -X POST "http://localhost:8080/api/v2/torrents/add" \
  --data-urlencode "urls=magnet:?xt=..."

Ver documentación: qBittorrent Web API

¿Cómo organizo automáticamente las descargas?¶

Opción 1: Categorías en qBittorrent

  1. Configurar categorías en qBittorrent Web UI
  2. Asignar carpetas de destino por categoría
  3. Al añadir torrents, especificar categoría

Opción 2: Script de organización

Usa el script de organización automática mencionado en Trucos
