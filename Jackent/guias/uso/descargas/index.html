Gestión de Descargas¶

Todo lo que necesitas saber para gestionar tus descargas en qBittorrent desde Jackent.

Añadir Torrents¶

Opción 1: Desde los Resultados¶

  1. Buscar contenido
  2. Clic en "Añadir a qBittorrent"
  3. El torrent se añade automáticamente
  4. La descarga comienza inmediatamente

Gestionar Descargas en qBittorrent¶

Acceder a qBittorrent Web UI¶

http://localhost:8080

Credenciales por defecto:

  • Usuario: admin
  • Contraseña: adminadmin

Funciones Principales¶

Función Descripción
Ver progreso Porcentaje de descarga en tiempo real
Pausar/Reanudar Control manual de descargas
Prioridad Cambiar orden de descarga
Límites de velocidad Configurar límites de subida/bajada
Eliminar torrents Borrar torrents (con o sin archivos)

Organización de Archivos¶

Carpeta de Descargas¶

Todos los archivos se guardan en:

./Descargas/

Esta carpeta está montada fuera del contenedor Docker, por lo que:

  • ✅ Los archivos persisten aunque reinicies Docker
  • ✅ Puedes acceder a ellos directamente desde tu sistema
  • ✅ Fácil integración con Jellyfin, Plex o Kodi

Estructura Automática¶

Cada torrent se guarda en su propia carpeta:

Descargas/
├── The.Matrix.1999.1080p.BluRay/
│   └── The.Matrix.1999.1080p.BluRay.mkv
├── Breaking.Bad.S01.1080p/
│   ├── Breaking.Bad.S01E01.mkv
│   ├── Breaking.Bad.S01E02.mkv
│   └── ...
└── ...

Control y Configuración¶

Ver Estado de Torrents¶

Los torrents pueden estar en diferentes estados:

Estado Descripción
Descargando Descarga activa en progreso
En seed Descarga completa, compartiendo
Pausado Descarga detenida manualmente
En cola Esperando para iniciar
Error Problema con el torrent

Configuración Avanzada¶

Desde Jackent puedes configurar:

Auto-eliminación:

  • Elimina torrents completados después de X horas
  • Útil para ahorrar espacio

Límite de espacio:

  • Define límite máximo de almacenamiento
  • Elimina torrents antiguos automáticamente

Acceso a configuración:

  1. Clic en el botón ⚙️ (Configuración)
  2. Ajustar parámetros
  3. Guardar cambios

Integración con Media Servers¶

Jellyfin / Plex¶

  1. Configurar biblioteca apuntando a ./Descargas
  2. Jackent descarga el contenido
  3. Jellyfin/Plex detecta automáticamente los nuevos archivos
  4. Escanear biblioteca

Ejemplo de configuración:

Biblioteca de Películas: /path/to/Jackent/Descargas/Peliculas
Biblioteca de Series: /path/to/Jackent/Descargas/Series

Kodi¶

  1. Añadir fuente de video
  2. Navegar a la carpeta Descargas
  3. Kodi indexará el contenido automáticamente

Mejores Prácticas¶

Gestión de Espacio¶

✅ Habilitar auto-eliminación: Borra torrents antiguos automáticamente

✅ Configurar límite de espacio: Previene llenar el disco

✅ Monitorear uso: Revisa regularmente qBittorrent Web UI

Optimización de Descargas¶

Ratio de seed:

  • Contribuye a la comunidad
  • Configura ratio mínimo (ej: 1.0)
  • qBittorrent puede pausar automáticamente al alcanzarlo

Límites de velocidad:

  • Evita saturar tu conexión
  • Configura límites en qBittorrent Web UI
  • Programa horarios (descarga rápida de noche)

Prioridad de descargas:

  • Marca torrents importantes como "alta prioridad"
  • Los demás descargarán después

Solución de Problemas¶

Descarga muy lenta¶

Causas:

  • Pocos seeders
  • Límites de velocidad configurados
  • Problemas con tu ISP

Soluciones:

  1. Verificar número de seeders (elegir torrents con más seeds)
  2. Revisar límites en qBittorrent
  3. Comprobar puertos abiertos

Torrent no descarga¶

Causas:

  • Torrent muerto (sin seeders)
  • Problema de conexión
  • Error en el archivo .torrent

Soluciones:

  1. Buscar otra versión del mismo contenido
  2. Verificar conectividad en qBittorrent
  3. Reiniciar el torrent

No encuentro los archivos descargados¶

Ubicación: 

# Windows
C:\Users\TU_USUARIO\Desktop\Jackent\Descargas

# Linux/Mac
/home/usuario/Jackent/Descargas

Verificar:

  1. Navega a la carpeta del proyecto
  2. Busca la carpeta Descargas
  3. Los archivos están en subcarpetas con el nombre del torrent

Comandos Útiles¶

Ver espacio usado¶

# Windows
dir "Descargas" /s

# Linux/Mac  
du -sh Descargas/

Limpiar descargas antiguas¶

# Desde qBittorrent Web UI o manualmente:
rm -rf Descargas/nombre_del_torrent/

Ver logs de qBittorrent¶

docker compose logs qbittorrent
