GET /health¶

Verifica la conectividad con Jackett y qBittorrent.

Request¶

GET /health

Response (Success)¶

{
  "ok": true
}

Response (Error)¶

{
  "ok": false,
  "error": "Connection to Jackett failed"
}

Status Codes¶

  • 200 - Todos los servicios operativos
  • 500 - Error en Jackett o qBittorrent

Descripci√≥n¶

Este endpoint realiza verificaciones en tiempo real de:

  • ‚úÖ Conectividad con Jackett
  • ‚úÖ Validez de API Key de Jackett
  • ‚úÖ Conectividad con qBittorrent
  • ‚úÖ Autenticaci√≥n con qBittorrent Web UI

Uso Recomendado¶

// Verificar estado cada 30 segundos
async function checkHealth() {
  const response = await fetch('/api/health');
  const data = await response.json();

  if (data.ok) {
    updateIndicator('status', 'online');
  } else {
    updateIndicator('status', 'offline');
    console.error('Service error:', data.error);
  }
}

setInterval(checkHealth, 30000);
