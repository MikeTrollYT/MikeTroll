Preguntas Frecuentes - Uso¶

Respuestas a las preguntas más comunes sobre el uso de JackBrid.

General¶

¿Puedo usar Real-Debrid en lugar de AllDebrid?¶

Actualmente JackBrid solo soporta AllDebrid. El soporte para Real-Debrid está en consideración para futuras versiones.

¿Puedo usar JackBrid en móvil?¶

Sí, la interfaz es responsive. Funciona en:

  • ✅ Navegadores móviles (Chrome, Safari)
  • ✅ Tablets
  • ❌ No hay app nativa (aún)

¿Es necesario tener una cuenta Premium en AllDebrid?¶

Puedes usar una cuenta gratuita, pero tendrás limitaciones en:

  • Número de descargas diarias
  • Velocidad de descarga
  • Tamaño de archivos
  • Tiempo de expiración de links

Recomendación: Cuenta Premium para mejor experiencia.

Búsqueda¶

¿Por qué no aparecen resultados?¶

Posibles causas:

  1. Trackers no seleccionados - Marca al menos un tracker
  2. Jackett no configurado - Verifica que Jackett tenga trackers activos
  3. Búsqueda muy específica - Prueba términos más generales
  4. Trackers caídos - Algunos trackers pueden estar temporalmente inactivos

Solución: 

1. Verificar indicadores de estado (deben estar en verde)
2. Probar con nombre en inglés
3. Simplificar búsqueda (solo título, sin año ni calidad)

¿Cómo busco series por episodio específico?¶

Usa el formato estándar:

Breaking Bad S01E01        # Temporada 1, Episodio 1
Game of Thrones S05E09     # Temporada 5, Episodio 9
The Office US S02          # Temporada 2 completa

¿Qué trackers son los mejores?¶

Depende del contenido:

Tipo Trackers Recomendados
Películas 1337x, YTS, RARBG
Series EZTV, ThePirateBay
Anime Nyaa, AnimeTosho
Música Rutracker, ThePirateBay
Software 1337x (solo open source)
Documentales MVGroup, DocsPedia

AllDebrid¶

¿Cuánto tarda en procesar un torrent?¶

Depende del tamaño y la disponibilidad:

  • Archivos pequeños (< 1 GB): 10-30 segundos
  • Películas HD (1-5 GB): 30-60 segundos
  • Packs grandes (> 10 GB): 1-5 minutos

Si tarda más de 10 minutos, probablemente el torrent esté muerto (sin seeders).

¿Por qué algunos torrents se quedan en "En Cola"?¶

Causas comunes:

  1. Sin seeders - El torrent está muerto
  2. Torrent corrupto - Hash incorrecto
  3. Límite alcanzado - Has superado el límite diario de tu plan

Solución:

  • Busca otro torrent con más seeders (> 50)
  • Verifica tu límite en AllDebrid Dashboard
  • Elimina el torrent y prueba con otro

¿Cómo libero espacio en AllDebrid?¶

  1. Ve a la sección "Mis Elementos"
  2. Elimina torrents ya descargados o vistos
  3. AllDebrid limpia automáticamente contenido antiguo (según tu plan)

Tip: Descarga localmente el contenido importante y elimínalo de AllDebrid.

¿Los links de AllDebrid expiran?¶

Sí, los links directos expiran después de cierto tiempo (varía por plan).

Recomendación:

  • Descarga contenido importante localmente
  • No dependas 100% de la nube para archivos importantes
  • Renueva links si es necesario desde AllDebrid

Reproducción¶

¿Por qué no hay sonido en el video?¶

El reproductor web solo soporta ciertos códecs de audio:

  • ✅ AAC, MP3, Opus
  • ❌ AC3, DTS, TrueHD

Solución:

  1. Copiar link del video
  2. Abrir en VLC (soporta todos los códecs)
  3. VLC reproducirá el audio correctamente

¿Puedo cambiar los subtítulos?¶

Sí, el reproductor soporta subtítulos WebVTT (.vtt):

  1. Descargar subtítulos de OpenSubtitles
  2. Convertir .srt a .vtt (si es necesario)
  3. En el reproductor: Configuración → Cargar subtítulo

Herramientas de conversión: - Subtitle Edit - Conversores online (.srt a .vtt)

¿Por qué el video se ve entrecortado?¶

Causas:

  1. Conexión lenta - Insuficiente ancho de banda
  2. Calidad muy alta - 4K requiere > 50 Mbps
  3. Servidor saturado - Alta demanda en AllDebrid

Soluciones:

  • Reduce la calidad en configuración del reproductor
  • Descarga en lugar de streaming
  • Verifica tu velocidad de Internet: Fast.com
  • Prueba en horario con menos tráfico

¿Cómo reproduzco en mi Smart TV?¶

Método 1: VLC en TV 

1. Copiar link de AllDebrid
2. Abrir VLC en la TV
3. Red → Abrir flujo de red
4. Pegar link y reproducir

Método 2: Chromecast 

1. Reproducir en navegador Chrome (PC)
2. Clic en ícono de Chromecast
3. Seleccionar TV

Método 3: DLNA/UPnP 

1. Usar servidor DLNA (Plex, Jellyfin)
2. Añadir link como fuente externa
3. Reproducir desde la TV

Problemas Comunes¶

El indicador de estado está en rojo¶

Causas:

  1. Backend no iniciado - docker compose ps para verificar
  2. API Key incorrecta - Revisar .env
  3. Jackett no accesible - Verificar http://localhost:9117

Solución: 

# Verificar logs
docker compose logs backend

# Reiniciar servicios
docker compose restart backend

No puedo añadir torrents a AllDebrid¶

Verificar:

  1. ✅ API Key válida en .env
  2. ✅ Cuenta AllDebrid activa
  3. ✅ No has superado el límite diario
  4. ✅ El torrent tiene seeders

Test manual: 

curl "https://api.alldebrid.com/v4/user?agent=JackBrid&apikey=TU_API_KEY"

Si responde con datos de usuario, la API funciona correctamente.

La búsqueda es muy lenta¶

Optimizaciones:

  1. Reduce el número de trackers (máx 3-4)
  2. Limita resultados a 30-50
  3. Activa "Solo con Seeders"
  4. Verifica que Jackett esté actualizado

Privacidad y Seguridad¶

¿Es seguro usar JackBrid?¶

Sí, JackBrid es seguro:

  • ✅ No almacena contraseñas ni datos personales
  • ✅ API Keys solo se guardan localmente (.env)
  • ✅ AllDebrid anonimiza tu IP ante los trackers
  • ✅ No hay tracking ni analytics externos

¿Necesito usar VPN?¶

No es obligatorio, AllDebrid ya protege tu identidad ante los trackers.

Usa VPN si:

  • Tu ISP bloquea ciertos sitios
  • Quieres privacidad adicional
  • Normalmente usas VPN para todo

¿Puedo compartir links de AllDebrid?¶

No es recomendable:

  • Los links están asociados a tu cuenta
  • Pueden rastrearse
  • Expiran después de cierto tiempo
  • Compartir puede violar términos de servicio

Alternativa: Descarga localmente y comparte el archivo.

Avanzado¶

¿Cómo integro JackBrid con Plex/Jellyfin?¶

1. Buscar contenido en JackBrid
2. Añadir a AllDebrid
3. Copiar link directo
4. Descargar al servidor de medios:
   wget "https://...link..." -O "/media/movies/pelicula.mkv"
5. Escanear biblioteca en Plex/Jellyfin

¿Puedo automatizar descargas?¶

Sí, usando la API de AllDebrid:

# Script ejemplo
curl "https://api.alldebrid.com/v4/magnet/upload" \
  -d "agent=JackBrid" \
  -d "apikey=TU_API_KEY" \
  -d "magnets[]=magnet:?xt=..."

Ver documentación: AllDebrid API

¿Cómo descargo múltiples archivos en lote?¶

Usa aria2c:

# Crear archivo con links
nano links.txt
# (pegar links, uno por línea)

# Descargar con 16 conexiones
aria2c -x 16 -s 16 -i links.txt -d ~/Downloads/
