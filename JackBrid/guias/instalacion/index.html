Guía de Instalación de JackBrid¶

Esta guía te llevará paso a paso por el proceso de instalación y configuración de JackBrid en tu servidor local o remoto.

Tipos de Instalación¶

  • Instalación con Docker

    Instalación rápida y sencilla usando contenedores Docker. Recomendado para la mayoría de usuarios.

    Empezar instalación

  • Instalación Manual

    Instalación avanzada sin Docker. Para usuarios con conocimientos técnicos que prefieren control total.

    Instalación manual

  • Configuración Avanzada

    Personaliza puertos, acceso remoto, HTTPS y configuración de Nginx reverse proxy.

    Configuración avanzada

  • Mantenimiento

    Actualización, verificación del sistema y desinstalación de JackBrid.

    Mantenimiento

Requisitos del Sistema¶

  • CPU: 1 núcleo @ 1 GHz
  • RAM: 512 MB disponibles
  • Almacenamiento: 1 GB de espacio libre
  • Red: Conexión a Internet estable
  • CPU: 2+ núcleos @ 2 GHz
  • RAM: 2 GB disponibles
  • Almacenamiento: 5 GB+ (para descargas temporales)
  • Red: 10+ Mbps para streaming fluido
  • Docker 20.10 o superior
  • Docker Compose 2.0 o superior
  • Cuenta AllDebrid con API Key activa
  • Node.js 18.0 o superior
  • npm o yarn
  • Nginx (opcional, para producción)
  • Git para clonar el repositorio
  • Cuenta AllDebrid con API Key activa

Sistemas Operativos Compatibles

  • Linux (Ubuntu 20.04+, Debian 11+, Fedora 35+)
  • Windows 10/11 con WSL2
  • macOS 11+ (Big Sur o superior)
  • NAS Synology, QNAP, Unraid con Docker

Soporte¶

Si encuentras problemas durante la instalación:

  1. Revisar logs: 

    docker compose logs backend
docker compose logs jackett

  2. Consultar FAQ: FAQ

  3. Abrir un issue: GitHub Issues
