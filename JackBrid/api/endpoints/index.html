Documentación de API REST¶

Referencia completa de todos los endpoints disponibles en el backend de JackBrid.

Información General¶

Base URL¶

Desarrollo Local: 

http://localhost:3000

Producción (vía Nginx): 

http://localhost:8998/api

Content-Type¶

Todas las solicitudes y respuestas usan: 

Content-Type: application/json

CORS¶

El backend tiene CORS habilitado para todos los orígenes: 

Access-Control-Allow-Origin: *

Autenticación¶

Las API Keys de Jackett y AllDebrid se configuran mediante variables de entorno en el servidor. No se requiere autenticación desde el cliente para usar el API de JackBrid.

Endpoints por Categoría¶

  • Health & Status

    Verificación de estado del sistema, conectividad con servicios externos y URLs de acceso.

    Ver endpoints

  • Trackers

    Gestión y listado de trackers (indexers) configurados en Jackett.

    Ver endpoints

  • Búsqueda de Torrents

    Búsqueda avanzada de torrents en múltiples trackers con filtros y ordenamiento.

    Ver endpoints

  • Gestión de Torrents

    Descarga de archivos .torrent y extracción de magnet links.

    Ver endpoints

  • AllDebrid Integration

    Añadir, listar, eliminar y obtener enlaces de descarga desde AllDebrid.

    Ver endpoints

  • Ejemplos de Integración

    Ejemplos completos en JavaScript, Python, cURL, Node.js y React.

    Ver ejemplos

Flujo General de Uso¶

sequenceDiagram
    participant Cliente
    participant API
    participant Jackett
    participant AllDebrid

    Note over Cliente,AllDebrid: 1️⃣ Búsqueda
    Cliente->>API: GET /search?q=matrix
    API->>Jackett: Consultar trackers
    Jackett-->>API: Resultados
    API-->>Cliente: Lista de torrents

    Note over Cliente,AllDebrid: 2️⃣ Añadir
    Cliente->>API: POST /add {magnet}
    API->>AllDebrid: Upload magnet
    AllDebrid-->>API: ID procesado
    API-->>Cliente: Confirmación

    Note over Cliente,AllDebrid: 3️⃣ Obtener Links
    Cliente->>API: POST /download-links {id}
    API->>AllDebrid: Get CDN links
    AllDebrid-->>API: URLs directas
    API-->>Cliente: Links de video

Códigos de Estado HTTP¶

Código Significado Uso en JackBrid
200 OK Operación exitosa
400 Bad Request Parámetros faltantes o inválidos
500 Internal Server Error Error en Jackett o AllDebrid

Estructura de Error¶

{
  "error": "Descripción del error",
  "ok": false
}

Errores Comunes¶

Jackett no disponible: 

{
  "ok": false,
  "error": "Connection to Jackett failed"
}

API Key inválida: 

{
  "error": "Invalid API Key"
}

Tracker no configurado: 

{
  "results": [],
  "error": "Tracker 'example' not found"
}

Rate Limiting¶

Límites Recomendados¶

Endpoint Límite Sugerido Razón
/search 10 req/min Carga en Jackett
/add 20 req/min Límite de AllDebrid
/list 30 req/min Consulta simple
/download-links 30 req/min Consulta simple

AllDebrid Rate Limits¶

  • Free: ~10 requests/minuto
  • Premium: ~100 requests/minuto
